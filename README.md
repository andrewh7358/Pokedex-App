# Pokédex

This is a code demo of a Pokédex built with TypeScript and React.

This project uses the [PokéAPI](https://pokeapi.co/), an open-source RESTful API that provides data for the Pokémon main series games.

This Pokédex displays details for first-generation Pokémon, including their sprite, type, height, weight, and an in-game description. You can filter Pokémon by name or navigate directly to a specific entry using its Pokédex number.
